Quando procurar um otorrino?

Sinais de que você precisa de avaliação

Alguns sintomas podem indicar a necessidade de uma consulta com otorrinolaringologista. Não ignore sinais persistentes.

  • ✓ Nariz entupido frequentemente
  • ✓ Dor ou secreção no ouvido
  • ✓ Zumbido
  • ✓ Tontura recorrente
  • ✓ Dor de garganta constante
  • ✓ Ronco intenso
  • ✓ Dificuldade para respirar
  • ✓ Rouquidão persistente

Identificou algum sintoma?

Agende uma avaliação com nossa equipe especializada. Quanto antes o diagnóstico, melhor o prognóstico.
